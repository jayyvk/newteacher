# NewTeacher Submission Copy

## 1-Sentence Pitch

NewTeacher is a voice AI tutor that teaches from children's textbooks using RAG, then uses Firecrawl to bring in live context so lessons stay current, relevant, and adaptive.

## Short Description

NewTeacher turns static textbooks into an interactive voice teacher for children. It uses RAG over textbook content to give curriculum-grounded explanations, then uses Firecrawl to pull in live information when a student asks for current relevance or real-world examples. Powered by an ElevenLabs voice agent, it feels like talking to a patient teacher who explains, adapts, and connects lessons to the world as it changes.

## Longer Description

Children often learn from books that are accurate but static. NewTeacher solves that by combining three things: textbook-grounded retrieval, voice-native tutoring, and live web context.

First, it indexes textbook and curriculum material so answers stay aligned with what the child is learning. Then an ElevenLabs voice agent turns the experience into a spoken back-and-forth lesson instead of a chatbot. Finally, when the student asks a question that benefits from current information, Firecrawl fetches live context from the web so the lesson can connect old knowledge to the modern world.

For example, a student learning history can ask about an old war, then ask how conflicts still shape the world today. NewTeacher explains the original topic from the textbook, then enriches the answer with current context in age-appropriate language.

The result is a teacher that is not stuck in an outdated book. It adapts to the student, the lesson, and the world.

## Problem

Textbooks do not adapt to a child's pace, curiosity, or the present day. Students need explanations they can talk through, revisit, and connect to real life.

## Solution

NewTeacher is an AI teacher that:

- teaches from trusted textbook material
- speaks naturally through voice
- answers follow-up questions conversationally
- uses live web context when the lesson needs current relevance

## Why ElevenLabs + Firecrawl

- ElevenLabs makes the experience feel like a real spoken lesson instead of a text interface.
- Firecrawl makes the teacher aware of what is happening now, so lessons are not trapped in static content.

## Demo Script

1. Ask NewTeacher to explain a textbook topic.
2. Hear a simple spoken explanation grounded in the source material.
3. Ask a follow-up that requires modern relevance.
4. Show Firecrawl retrieving live context.
5. Hear NewTeacher connect the textbook lesson to today's world.
6. End with a recap or mini quiz.
